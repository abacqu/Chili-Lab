# Chili-Lab

A web application that allows users to view a collection of recipes that have been completed by the Chili Lab team. Users are able to submit recipes to be made by the team at a future date.

# Techonologies Used

HTML
CSS
JavaScript
MongoDB
Express
NodeJS
